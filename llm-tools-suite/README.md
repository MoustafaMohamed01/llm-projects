# LLM Tools Suite

An integrated collection of AI-powered tools for productivity, built with **Large Language Models**.

Available in two versions:

* **[v1-streamlit](./v1-streamlit/)** – Python + Streamlit app
* **[v2-web](./v2-web/)** – HTML/CSS/JS web app

---

## 🚀 Features

* 🧠 AI Assistant
* 📝 Blog AI Assistant
* 📊 CSV Analyzer
* 💻 SQL Query Generator
* 📄 Document Summarizer
* 🌐 Website Summarizer
* 🔍 Code Explainer
