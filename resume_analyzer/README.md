# AI Resume Analyzer

This repository contains the full source code for the **AI Resume Analyzer** project  an AI-powered application for evaluating resumes using advanced natural language processing.  

The project is organized into modular components to support scalability and clean development practices.

## Project Structure

```

resume_analyzer/
│
├── streamlit/           # Streamlit web interface for the Resume Analyzer
│
├── web/                 # (Coming soon) Web version for online deployment
│
└── README.md            # Main project overview

````

## Overview

The **AI Resume Analyzer** helps users assess resumes by:
- Extracting and analyzing resume content (PDF/DOCX)
- Evaluating skills and experience relevance
- Providing improvement feedback powered by AI models

## Tech Stack

- **Python 3.12+**
- **Streamlit**
- **Google Generative AI API**
- **PyPDF2**, **python-docx**

## Getting Started

To run the Streamlit interface locally:

```bash
cd streamlit
pip install -r requirements.txt
streamlit run app.py
````
